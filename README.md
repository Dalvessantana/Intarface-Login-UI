# Intarface-Login-UI

Uma interface de autenticação moderna, responsiva e de alta fidelidade visual, desenvolvida exclusivamente com **HTML** e **CSS**.

## 🎯 Objetivo
Este projeto foi criado para exercitar habilidades avançadas de estilização, focando em:
- Estruturação semântica.
- Flexbox para alinhamento e centralização.
- Design System baseado em uma paleta de cores esmeralda.
- Responsividade completa para dispositivos móveis.

## 🛠️ Tecnologias
![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) 
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

## ✨ Destaques
- **Sem Dependências:** Construído 100% com CSS puro, sem frameworks (como Bootstrap ou Tailwind).
- **Pixel Perfect:** Atenção aos detalhes de espaçamento, ícones e tipografia.
- **UI/UX:** Feedback visual em botões e links (estados de hover).
